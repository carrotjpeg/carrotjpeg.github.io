# Evana Thomson - Portfolio Website

Personal portfolio website showcasing projects, research, and skills. Built with vanilla HTML, CSS, and JavaScript.

## About

This is the personal website of Evana Thomson, a student studying EECS and Material Science at UC Berkeley. The site features:

- **About Me**: Introduction and background
- **Projects**: Portfolio of technical projects
- **Research**: Academic research and publications
- **Skills**: Technical skills and expertise

## Local Development

### Prerequisites

- Node.js (for running local server)

### Getting Started

1. Clone the repository:
```bash
git clone https://github.com/carrotjpeg/carrotjpeg.github.io.git
cd carrotjpeg.github.io
```

2. Install dependencies (optional, only needed for development server):
```bash
npm install
```

3. Start the local development server:
```bash
npm start
```

This will start a local server at `http://localhost:3000` and automatically open it in your browser.

Alternatively, you can run the dev server without auto-opening:
```bash
npm run dev
```

## Deployment
